Nolan Showcase Theme 05 is a classic WordPress theme for a modern web design and website services company. It uses PHP templates, local CSS, local JavaScript, and local SVG imagery only.

## Included

- Classic WordPress templates for home, pages, posts, archives, search, 404, comments, sidebar, and search form.
- Local responsive stylesheet at `assets/css/theme.css`.
- Vanilla JavaScript navigation behavior at `assets/js/theme.js`.
- Local SVG sample imagery in `assets/images/`.
- Static GitHub Pages mirror in `docs/themes/nolan-showcase-theme-05/`.

## Notes

Forms are presentation-ready for Gravity Forms, WPForms, Contact Form 7, or a custom WordPress handler. The theme intentionally does not submit data to external services or APIs.
