# Nolan Showcase Theme

This is a classic WordPress portfolio theme for a software developer and web developer.

## Included files

- `style.css`
- `functions.php`
- `index.php`
- `header.php`
- `footer.php`
- `front-page.php`
- `page.php`
- `single.php`
- `archive.php`
- `404.php`
- `assets/css/theme.css`
- `assets/js/theme.js`

## Notes

- Uses local assets only.
- Uses a sticky header and responsive navigation.
- Uses vanilla JavaScript for menu toggling and reveal animations.
- Requires activation in WordPress through Appearance > Themes.
