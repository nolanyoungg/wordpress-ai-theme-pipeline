# Nolan Showcase Theme X12

Classic WordPress theme for a premium WordPress design, maintenance, AI integration, WooCommerce, SEO, analytics, and digital services company.

## Included

- Complete classic theme templates.
- Accessible header with desktop dropdowns and mobile drawer.
- Local CSS, JavaScript, and SVG image assets.
- Styled quote/contact form markup in the static preview and CSS support for WordPress form plugins.
- No API keys, CDN dependencies, or live AI integrations.

## Notes

AI chatbot integration and AI website automation are presented as service offerings only. Backend form processing is intentionally left to Gravity Forms, WPForms, Contact Form 7, or a similar WordPress plugin.
