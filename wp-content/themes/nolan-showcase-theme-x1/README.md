# Nolan Showcase Theme X1

This is a classic WordPress portfolio theme intended to present a premium, dark, high-tech developer portfolio focused on:

- Software development
- WordPress + WooCommerce
- Automation
- Analytics and tracking
- AI integration
- Technical leadership

## Install

1. Upload the zip produced by GitHub Actions (`nolan-showcase-theme-x1.zip`) from the workflow artifacts.
2. In WordPress Admin, activate **Nolan Showcase Theme X1**.
3. Set a static front page to use the homepage sections in `front-page.php`.

## Files

Core theme templates live directly in the theme folder. Shared post/page templates live in `template-parts/`. Theme CSS/JS lives under `assets/`.

## Static preview

This theme has a matching static preview for GitHub Pages under `docs/themes/nolan-showcase-theme-x1/`.
