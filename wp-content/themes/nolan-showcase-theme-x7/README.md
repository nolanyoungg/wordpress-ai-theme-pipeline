# Nolan Showcase Theme X7 (MNY Photo)

Classic WordPress theme for a premium photography business brand (“MNY Photo”). This version focuses on a warm-luxury, cinematic editorial feel and a custom navigation system with rich desktop dropdown panels plus a mobile off-canvas menu.

## Key features
- Classic PHP templates (no block theme)
- No external runtime assets/CDNs
- Custom Nolan menu system (keyboard accessible, aria updates, Escape + click-outside close)
- Conversion-focused front page with 13 sections
- Matching GitHub Pages static preview in `docs/themes/nolan-showcase-theme-x7/`

## Dev notes
Compiled assets are committed:
- `assets/css/main.css`
- `assets/js/main.min.js`

Source assets:
- SCSS entry: `scss/main.scss`
- JS source: `assets/js/main.js`

Build tooling lives in `build/` (Gulp only).
