# Nolan Showcase Theme X13

## Theme Overview

Nolan Showcase Theme X13 is a classic WordPress theme for a fictional premium national WordPress agency. It presents design, development, maintenance, support, WooCommerce, SEO, analytics, performance, AI chatbot integration, and AI website automation services in clear business language.

## Folder Structure

- `style.css`: WordPress theme header.
- `functions.php`: theme support, menus, sidebar, and asset enqueueing.
- `header.php` / `footer.php`: logo, navigation, CTA, landmarks, and footer content.
- `front-page.php`: custom agency homepage.
- `page.php`, `single.php`, `archive.php`, `search.php`, `404.php`, `comments.php`, `searchform.php`, `sidebar.php`: classic WordPress templates.
- `template-parts/`: reusable content partials.
- `assets/css/theme.css`: responsive production CSS.
- `assets/js/theme.js`: vanilla JavaScript navigation behavior.
- `assets/images/`: local SVG visual assets.

## Features

- Premium agency homepage with hero, stats, services, process, work examples, support, AI services, testimonial, FAQ, and final CTA.
- Service-hub support for the `what-we-do` page.
- Quote-ready contact form pattern for Gravity Forms, WPForms, or Contact Form 7 replacement.
- Solid desktop dropdowns and accessible mobile drawer.
- Local CSS, JavaScript, and image assets only.

## Page Templates

The theme uses standard classic templates. Create WordPress pages such as Home, Who We Are, What We Do, Work, Resources, and Contact, then assign a static front page in Settings > Reading.

## Header Behavior

The header follows `Logo | Main Navigation | Get a Quote CTA`. Desktop dropdowns open on hover and keyboard focus with solid backgrounds. Mobile navigation uses a hamburger button with `aria-expanded`, `aria-controls`, Escape close, and no focus trap.

## Static Preview Notes

The matching GitHub Pages preview lives in `docs/themes/nolan-showcase-theme-x13/`. It mirrors the visual direction and navigation behavior in static HTML, CSS, and vanilla JavaScript.

## Accessibility Notes

The theme includes a skip link, semantic landmarks, visible focus states, accessible form labels, keyboard-friendly navigation, readable contrast, and reduced-motion support.

## Runtime Dependencies

There are no external runtime dependencies, CDNs, API keys, remote fonts, paid APIs, OpenAI API usage, or frontend frameworks.

## Activation Instructions

Upload or install `nolan-showcase-theme-x13.zip` through WordPress, activate the theme, create the recommended pages, assign the Primary and Footer menus, and set a static homepage.

## ZIP / Build Notes

The installable archive is expected at `zippedTheme/nolan-showcase-theme-x13.zip`. Regenerate it whenever files under `wp-content/themes/nolan-showcase-theme-x13/` change.
